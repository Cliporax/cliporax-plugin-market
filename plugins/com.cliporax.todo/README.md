# TODO

Adds an independent TODO list tab to Cliporax.

- Add local TODO items from the content tab.
- Blank items are ignored after trimming whitespace.
- Duplicate items are rejected using trimmed, case-insensitive text.
- Leading and trailing whitespace is trimmed before display.
- Completed items move to the end of the list.
- Data is stored locally in browser storage and does not require backend IPC.
